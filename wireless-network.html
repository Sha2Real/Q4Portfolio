Wireless Networking

A wireless networrk is a computer network that makes use of Radio Frequency (RF) connections between nodes in the network.

Hardware

The core hardware consists of a WiFi router, modem, and ethernet cabling (e.g. Cat 5 or Cat 6 cable). This is the minimum equipment required to set up a wireless network.

Pros

  • Installing Wireless Network is faster
  • Requires One Time investment

Con

  • Less Security
  • Less Speed than wired Network

    • Wireless Summary

    A wireless network connects computers without using network cables. Computers use radio communications to send data between each other. You can communicate directly with other wireless computers, or connect to an existing network through a wireless AP.
